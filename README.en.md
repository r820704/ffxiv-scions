# FFXIV Traditional Chinese Server Tools

A toolkit for FINAL FANTASY XIV Traditional Chinese server players.

[繁體中文版本](README.md)

## Features

- Eureka weather forecast
- Logos action management
- Eurekan gear chains

## Tech Stack

- React 18 + TypeScript
- Vite
- Deployed via GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Tests

```bash
npm run test
```

## Build

```bash
npm run build
```

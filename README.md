# FFXIV 繁中伺服器工具

為 Final Fantasy XIV 繁體中文伺服器玩家打造的工具網站。

## 功能

- 天氣查詢工具：查詢艾歐乍亞各區域的天氣預報

## 技術棧

- React 18 + TypeScript
- Vite
- GitHub Pages 部署

## 開發

```bash
npm install
npm run dev
```

## 測試

```bash
npm run test
```

## 建構

```bash
npm run build
```

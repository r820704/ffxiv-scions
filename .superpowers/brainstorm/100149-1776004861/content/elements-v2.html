像素風元素 Icon

每個 icon 用高對比色繪製，深色底 + 明亮圖案，確保清楚可辨

這些是概念示意，實際 icon 會更精緻。重點是：明亮圖案 + 深色背景 = 高對比，不會跟底色混在一起。

用法一：卡片左側元素邊框 + 微光

只用 4px 細邊框 + 柔和光暈，不是整塊填色
